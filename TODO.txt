### 4. Add sound effects for each game action/animation

**Context:** The game currently has no audio feedback. Sounds should be generated programmatically using the Web Audio API (no audio files),
triggered in sync with each animation event.

**Requirements:**
- All sounds must be generated in code using `AudioContext` (no external audio files).
- Each action type should have a distinct sound:
  - `move` — Short, low thud (heavy marble rolling), example already provided in original TODO.
  - `enter` — A brighter, slightly longer sound (marble entering the board).
  - `capture` — An impact sound (marble knocked off the board).
  - `swap` — A double-hit or "whoosh" sound.
  - `promote` — A soft chime or ascending tone (marble reaching the finish zone).
  - `discard` / `pass` — A subtle card-swipe or neutral tick.
  - New turn banner — A short fanfare or notification sound.

**Implementation steps:**
1. Create a `SoundService` (or a standalone utility file) in `frontend/src/app/home/services/` that exposes one function per action type (e.g. `playMoveSound()`, `playCaptureSound()`, etc.).
2. Use `AudioContext`, `OscillatorNode`, and `GainNode` to synthesize each sound. See the reference implementation in the original TODO for the `move` sound as a starting template.
3. Trigger each sound from `board.component.ts` at the start of the corresponding marble animation (inside the animation playback logic keyed by `ActionType`).
4. Trigger the new-turn sound from `home.page.ts` when the new-turn banner is shown.

**Reference implementation for `move` sound (from original TODO):**
```typescript
const playMoveSound = () => {
  const audioCtx = new (window.AudioContext || (window as any).webkitAudioContext)();
  const oscillator = audioCtx.createOscillator();
  const gainNode = audioCtx.createGain();

  oscillator.type = 'triangle';
  oscillator.frequency.setValueAtTime(150, audioCtx.currentTime);
  oscillator.frequency.exponentialRampToValueAtTime(40, audioCtx.currentTime + 0.1);

  gainNode.gain.setValueAtTime(0.5, audioCtx.currentTime);
  gainNode.gain.exponentialRampToValueAtTime(0.01, audioCtx.currentTime + 0.1);

  oscillator.connect(gainNode);
  gainNode.connect(audioCtx.destination);

  oscillator.start();
  oscillator.stop(audioCtx.currentTime + 0.1);
};
```

**Relevant code:**
- `frontend/src/app/home/components/board/board.component.ts` — marble animation logic (action type → animation dispatch)
- `frontend/src/app/home/home.page.ts` — new-turn banner display logic
- `packages/shared/src/constants.ts` — `MARBLE_ANIMATION_DURATIONS` (use these as timing reference)
